Wiktoria Byra, nr ind.: 250131. Języki formalne i teoria translacji, laboratorium, lista 2.

Przed uruchomieniem programów należy skompilować je poleceniem:
make

Zad. 1
Program likwiduje nadmiarowe białe znaki, puste linie oraz zlicza liczbe słów i linii.
Sposób uruchomienia:
./zad1 < plik.txt
gdzie "plik.txt" będzie dowolnym plikiem telstowym, który chcemy wczytać.

Zad. 2
Program usuwa komentarze z kodu w języku python.
Sposób uruchomienia:
./zad2 < plik.py
gdzie "plik.py" będzie dowolnym plikiem z kodem w języku python, który chcemy wczytać.

Zad. 3
Program usuwa komentarze z kodu w języku c++ z możliwością ominięcia komentarzy dokumentacyjnych.
Sposób uruchomienia:
./zad2 < plik.cpp
LUB jeżeli chcemy zachować komentarze dokumentacyjne:
./zad2 < plik.cpp --doc
gdzie "plik.cpp" będzie dowolnym plikiem z kodem w języku c++, który chcemy wczytać.

Zad. 4
Program jest prostym kalkulatorem w odwrotnej notacji polskiej. 
Sposób uruchomienia:
./zad4
w kolejnych linijkach wpisujemy wyrażenia do obliczenia i zatwierdzamy je znakiem nowej linii.
